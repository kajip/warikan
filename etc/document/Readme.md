## 割り勘アルゴリズム

* *支払割合*を、普通の人の何倍払うかで表現
* *請求金額*を、*参加者*全員の*支払割合*の合計(*支払割合合計*)で割り、普通の人が払う金額(*支払基準額*)を計算する
* *支払基準額*に、各参加者の*支払割合*をかけて、その人の*支払額*を計算する
